Date Ranges add complexity. In this example, I try to provide an implementation that fulfils the following requirements.

  • Use standard Bootstrap 4 functionality - no third party libraries.
  • Date picker will disable start dates after the end date, if end date is set.
  • Date picker will disable end dates less than the start date, if start date is set.
  • Date picker will allow the user to select start/end dates in any order.
  • If, when picking the start date, the end dates is before the start date then the end date will be cleared.
  • Able to specify a minimum time duration in minutes and have the date picker enforce the duration for end dates.
  • Optionally: disable selecting dates in the past...local time only.

Of course, your requirements are likely different but hopefully this example demonstrates that complex functionality can be implemented easily with this component.
