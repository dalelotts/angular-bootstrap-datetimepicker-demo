Date Ranges add a more complexity. In this example, I try to provide an implementation that fulfils the following requirements.

  • Use standard Bootstrap 4 functionality - no third party libraries.
  • Start date must be before end date.
  • End date must be equal to or greater than start date.
  • The user is able to select start/end dates in any order.
  • If, when selecting the start date, the end dates is before the start date then the end date will be cleared.
  • Able to specify a minimum time duration in minutes.
  • Optionally: disable selecting dates in the past...local time only.

Of course, your requirements are likely different but hopefully this example demonstrates that event complex functionality can be implemented easily with this component.
