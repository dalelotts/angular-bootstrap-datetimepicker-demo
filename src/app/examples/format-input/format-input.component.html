See moment's parsing multiple formats page for more information on how these date formats are used.

The dlDateTimeInput directive allows the user to enter dates, in multiple formats, using only the keyboard.

Regardless of the format used to enter the date, the display format is always the same. In this example, the dates is displayed using "{{displayFormat}}" regardless of the format used to enter the date.

The supported input formats are defined in DL_DATE_TIME_INPUT_FORMATS.

The display format is defined in DL_DATE_TIME_DISPLAY_FORMAT.
